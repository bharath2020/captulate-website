Privacy Policy for Captulate

Last updated: January 5, 2025

Overview

Captulate respects your privacy. This policy explains how our app handles information when you use it to perform mathematical calculations from screen content.

How Captulate Works

Captulate uses screen recording permission to:

  • Capture user-selected portions of your screen
  • Extract numerical information from the captured area using on-device OCR
  • Compute mathematical calculations based on the extracted numbers
  • Display results instantly

Data Collection

Analytics Data

Captulate collects anonymous usage statistics to improve the app:

  • Feature usage frequency (e.g., number of calculations performed)
  • App performance metrics
  • Error occurrences (without screen content)
  • General usage patterns

We do NOT collect:

  • Personal identifying information
  • Screen content or captured images
  • Calculation values or results
  • Location data
  • Device identifiers that can identify you

Screen Content

  • Screen captures are processed locally and immediately discarded
  • No screen content is ever transmitted or stored

Screen Recording Permission

We request screen recording permission solely to:

  • Capture the specific screen area you select
  • Extract numerical values from that area
  • All capture and processing happens locally on your Mac

Privacy Principles

  • Anonymous Analytics Only: Usage data cannot be traced back to you
  • Local Processing: All screen capture and calculations happen on-device
  • No Personal Data: We never collect names, emails, or identifying information
  • Temporary Processing: Captured content exists only in memory during calculation

Your Control

  • Screen recording permission can be revoked in System Settings
  • Analytics help us improve Captulate but contain no personal information
  • All screen capture requires your explicit action
